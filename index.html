Methodology

How ForecastRank works

ForecastRank records each model’s forecast maximum for the following local calendar day at a fixed time. After that day has passed, the stored forecast is compared with the official daily maximum reported for Toronto Pearson International Airport.

The signed error is forecast minus observed temperature. A positive value means the forecast was too warm; a negative value means it was too cool. Ranking is based on the absolute error.

Forecast model data are delivered through Open-Meteo. Observed maximum-temperature data are supplied by Environment and Climate Change Canada. ForecastRank compares numerical model output, not necessarily the human-edited public forecast issued by each agency.
